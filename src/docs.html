API Documentation

Main Proxy Endpoint

GET /proxy

Parameters:

url (required)

The media URL to proxy (URL encoded)

headers (optional)

JSON string of headers (URL encoded). Example: {"referer": "https://example.com"}

Example:

https://your-domain.vercel.app/proxy?url=https%3A%2F%2Fexample.com%2Fvideo.m3u8&headers=%7B%22referer%22%3A%22https%3A%2F%2Fexample.com%22%7D

Embed Player Endpoint

GET /stream/s-2/{episode-id}/{language}

Parameters:

{episode-id} (required)

The anime episode ID from your source

{language} (required)

Language version: sub or dub

Example Embed:

<iframe src="https://your-domain.vercel.app/stream/s-2/136197/dub" width="100%" height="500px" frameborder="0" scrolling="no" allowfullscreen> </iframe>
